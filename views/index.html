collapseURL

URL Shortener App - One of the Free Code Camp Backend Challenges

Purpose: You can pass a URL you want to shorten as a parameter like this

https://collapse-url.glitch.me/new/[your-link]

You don't need the brackets around the link. You will then receive a shortened URL in the JSON response.

Example usage

https://collapse-url.glitch.me/new/https://www.google.com
https://collapse-url.glitch.me/new/http://www.example.com

Example output

{ "original_url": http://www.example.com, "short_url": "https://collapse-url.glitch.me/8170"}

Usage

Visit https://collapse-url.glitch.me/8170 and you will be redirected to http://www.example.com
